# PYTHON_GAME_SPACEWAR
Simple 2-d game using python(pygame).

# How to run
You need visual code studio or any IDE.
Press play button and play.
# INSTRUCTION
Yellow ship- W S A D shoot- LCTRL
Red ship- UP DOWN LEFT RIGHT shoot- RCTRL
